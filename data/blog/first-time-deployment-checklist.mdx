## Some background
My projects use Next.js and Supabase. Then are deployed to Vercel.

## The Checklist
- Set up supabase for production
- Sync local database with the production one
- Set up mailer lite account
- Set up plausible
- Set up google search console
- Setup a domain alias for gmail
- Link vercel deploy to github
- Set up domain
